## 1과목: 빅데이터 분석기획

| 키워드 | 암기 |
|--------|------|
| **3V** | Volume(양), Velocity(속도), Variety(다양성) |
| **5V** | + Veracity(신뢰성), Value(가치) |
| **데이터 유형** | 정형(DB,CSV) / 반정형(JSON,XML) / 비정형(텍스트,이미지) |
| **Hadoop** | HDFS(분산파일), MapReduce(분산처리), Hive(SQL), Spark(인메모리) |
| **NoSQL** | Redis(키-값), MongoDB(문서), HBase(컬럼), Neo4j(그래프) |
| **가명 vs 익명** | 가명=추가정보 없으면 식별불가 / 익명=복원불가 |
| **분석 절차** | 문제정의→수집→전처리→탐색·모델링→평가·해석 |

## 2과목: 빅데이터 탐색

| 키워드 | 암기 |
|--------|------|
| **결측 처리** | 삭제, 대체(평균/중앙/최빈), 예측(KNN·회귀) |
| **이상치 IQR** | 하한=Q1−1.5×IQR, 상한=Q3+1.5×IQR |
| **IQR** | Q3−Q1 |
| **중심경향** | 평균(이상치 민감), 중앙값(강건), 최빈값(범주형) |
| **가설검정** | t(평균비교), F(분산), χ²(독립성·적합도) |
| **p-value** | p<0.05 → 귀무가설 기각(유의미) |
| **상관 r** | −1~+1, 상관≠인과 |

## 3과목: 빅데이터 모델링

| 키워드 | 암기 |
|--------|------|
| **지도 vs 비지도** | 지도(레이블O, 회귀·분류) / 비지도(레이블X, 군집·차원축소) |
| **K-means** | 비지도, 군집화 |
| **PCA** | 비지도, 차원축소 |
| **로지스틱** | 이진분류, 확률 출력, 로짓=log(p/(1−p)) |
| **선형회귀** | RMSE, MAE, R² (F1은 분류!) |
| **의사결정나무** | 지니, 엔트로피, 정보이득 (R² 아님) |
| **Random Forest** | 배깅 |
| **XGBoost, LightGBM** | 부스팅 |
| **SVM** | 마진 최대화 |
| **KNN** | 이웃 k개 투표, 비모수 |
| **CNN** | 이미지 |
| **RNN** | 시계열, 텍스트 |

## 4과목: 빅데이터 결과 해석

### 분류 지표

| 지표 | 공식 | 용도 |
|------|------|------|
| **정확도** | (TP+TN)/전체 | 전체 맞춘 비율 |
| **정밀도** | TP/(TP+FP) | FP 비용 클 때 |
| **재현율** | TP/(TP+FN) | FN 비용 클 때(암검진) |
| **F1** | 2×P×R/(P+R) | 조화평균 |
| **AUC** | ROC 아래 면적 | 1에 가까울수록 좋음 |

### 회귀 지표

| 지표 | 특징 |
|------|------|
| **MSE** | 제곱, 이상치 민감 |
| **RMSE** | √MSE, 단위 동일 |
| **MAE** | 이상치 강건 |
| **R²** | 0~1, 설명력 |

### 오즈비(OR)

- OR>1: 변수↑ → 사건 확률↑
- OR<1: 변수↑ → 사건 확률↓

| 관계 | 산점도 |
| 시계열 | 시계열 그래프 |

## 자주 나오는 함정

- **3V에 Variance 없음** → Variance(X), Variety(O)
- **이상치 제거 ≠ 결측 처리**
- **F1 = 조화평균** (산술·기하 아님)
- **p>0.05 = 유의미하지 않음**
- **상관 ≠ 인과**
- **로지스틱 회귀 = 지도학습** (비지도 아님)
