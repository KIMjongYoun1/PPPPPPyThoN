## scikit-learn — 전처리

`from sklearn.preprocessing import LabelEncoder`

| 이름 | 읽는 법 | 기능 | 사용처 |
|------|---------|------|--------|
| **LabelEncoder** | 라벨 인코더 | 문자 → 정수 | 범주 **한 열** |
| **.fit** | 핏 | 규칙 학습 | train만 |
| **.transform** | 트랜스폼 | 규칙 적용 | test (fit 금지) |
| **fit_transform** | 핏 트랜스폼 | 둘 다 | train 편의 |

### LabelEncoder 예제

```python
from sklearn.preprocessing import LabelEncoder

cat_cols = X_train.select_dtypes(include=["object"]).columns.tolist()
for col in cat_cols:
    le = LabelEncoder()
    X_train[col] = le.fit_transform(X_train[col].astype(str))
    X_test[col] = le.transform(X_test[col].astype(str))
```

## SciPy — `from scipy import stats`

| 이름 | 읽는 법 | 기능 | 사용처 |
|------|---------|------|--------|
| **stats** | 스탯스 | 통계 함수 | 3유형 |
| **ttest_ind** | 티테스트 인디 | 독립 t검정 | 두 그룹 평균 |

### SciPy 예제

```python
from scipy import stats

a = df.loc[df["group"] == "A", "score"]
b = df.loc[df["group"] == "B", "score"]
t_stat, p_value = stats.ttest_ind(a, b, equal_var=False)  # 웰치 등 문제 조건에 맞게
```

## statsmodels — `from statsmodels.formula.api import ols, logit`

| 이름 | 읽는 법 | 기능 | 사용처 |
|------|---------|------|--------|
| **ols** | 오엘에스 | 선형회귀 | 다중선형, R², 계수 |
| **logit** | 로짓 | 로지스틱 | 오즈비, 이진 |
| **.fit** | 핏 | 적합 | 결과에 `.summary()` 등 |

### statsmodels 예제

```python
import pandas as pd
from statsmodels.formula.api import ols, logit

# 다중선형회귀: R², 회귀계수
model = ols("price ~ area + rooms + age", data=df).fit()
print(model.rsquared)
print(model.params)

# 로지스틱: 계수, 오즈비 exp(계수)
m = logit("left ~ satisfaction + years", data=df).fit()
print(m.params)
import numpy as np
print(np.exp(m.params))  # 오즈비
```
