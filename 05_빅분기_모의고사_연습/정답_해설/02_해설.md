# 02. Churn 이탈 예측 — 해설

## 풀이 흐름

1. **Step 4**: `Exited` → y, `CustomerId` 제거 후 X 분리
2. **Step 5**: `Geography`, `Gender` 범주형 → LabelEncoder
3. **Step 6**: RandomForestClassifier 학습
4. **Step 7**: 예측 → submission.csv (CustomerId, Exited)

## 핵심 포인트

- **범주형 인코딩**: `LabelEncoder` 사용 시 `fit`은 train에만, `transform`은 test에도 적용
- **test에 없는 범주**: `astype(str)`로 변환 후 인코딩하면 새 값도 처리 가능
- **대안**: `pd.get_dummies()` 사용 시 train/test 컬럼 맞춰야 함

## 주의사항

- `CustomerId`는 ID이므로 모델 입력에서 제거
- AUC/확률 제출 시: `model.predict_proba(X_test)[:, 1]` 사용
