# 05. YouTube 인기동영상 — 해설

## 풀이 흐름

- **Q1**: `channelId.value_counts().head(10)` → 해당 채널의 channelTitle
- **Q2**: `df.likes < df.dislikes` 필터 → channelTitle.unique()
- **Q3**: channelId별 channelTitle 개수 > 1 이면 채널명 변경
- **Q4**: `trending_date2` → dt.day_name() == "Sunday" → categoryId.value_counts()
- **Q5**: groupby(요일, categoryId) → pivot
- **Q6**: ratio = comment_count/view_count, view_count≠0, sort descending
- **Q7**: ratio≠0 중 최소
- **Q8**: dislikes/likes 최소 (like, dislike ≠ 0)
- **Q9**: channelId.value_counts().index[0] → 해당 채널명
- **Q10**: (title, channelId) value_counts ≥ 20 개수

## 핵심 포인트

- **value_counts()**: 빈도 집계, `.index`로 값 추출
- **dt.day_name()**: datetime → 요일명
- **pivot**: groupby 결과를 행/열 구조로 변환

## 주의사항

- `index_col=0`: DataManim 원본과 동일 (첫 컬럼이 index)
- view_count=0 제외 시 division by zero 방지
