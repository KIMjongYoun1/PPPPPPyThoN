# 24. 로지스틱 회귀 이직 예측 — 해설

## 풀이 흐름

1. X_train, y_train 로드, 수치형 변수만 선택
2. **statsmodels logit** (sklearn 아님)
3. **Q1 오즈비**: np.exp(회귀계수)
4. **Q2 유의한 변수**: pvalues < 0.05, Intercept 제외

## 핵심 포인트

- **logit**: statsmodels.formula.api
- **오즈비**: 계수가 1 증가할 때 오즈가 exp(β)배
- **formula**: "target ~ x1 + x2" (R 스타일)
