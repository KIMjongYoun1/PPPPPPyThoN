# 23. 가설검정 F검정, t검정 — 해설

## 풀이 흐름

1. 두 집단 분리, 로그 변환
2. **Q1-1**: F검정 — 분산 비 (큰 분산/작은 분산), ddof=1
3. **Q1-2**: 합동 분산 = ((n1-1)*var1 + (n2-1)*var2) / (n1+n2-2)
4. **Q1-3**: stats.ttest_ind(독립표본 t검정)

## 핵심 포인트

- **ddof=1**: 표본 분산 (자유도 n-1)
- **F검정**: 분자 > 분모 조건 — 등분산 검정
- **round(값, 3)**: 소수 셋째 자리
