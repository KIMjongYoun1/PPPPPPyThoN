# 09. Cancer 유방암 — 해설

## 풀이 흐름

1. X/y 로드, ID 분리
2. **정답 열은 `diagnosis` (M/B 문자열)** — `target` 열 없음. 모델용으로 `LabelEncoder`로 숫자화
3. 결측치 처리 (X는 수치형만 — **범주형 문자열 열이 없어** X용 `LabelEncoder` 루프 생략 가능)
4. RandomForestClassifier
5. submission (ID, **target** 컬럼명으로 예측값 저장 — 관례)

## 핵심 포인트

- GitHub `datarepo/cancer/y_train.csv` 헤더: `ID,diagnosis` → **`y_train["target"]` 쓰면 KeyError**
- 이진분류 (M/B); `RandomForestClassifier`는 숫자 라벨에 `fit`
- 정확도 비교 시 `y_test["diagnosis"]`도 같은 `LabelEncoder`로 `transform` 후 `pred`와 비교
