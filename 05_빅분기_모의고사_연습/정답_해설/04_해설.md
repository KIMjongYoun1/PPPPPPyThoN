# 04. HRdata 이직 예측 — 해설

## 풀이 흐름

1. **Step 4**: `enrollee_id` 분리, y_train = `target`
2. **Step 5**: object 타입 컬럼 자동 탐지 → LabelEncoder
3. **Step 6**: RandomForestClassifier
4. **Step 7**: 예측 → submission (enrollee_id, target)

## 핵심 포인트

- **select_dtypes(include=["object"])**: 범주형 컬럼 자동 탐지
- **y_test 존재**: `(pred == y_test["target"].values).mean()` 로 accuracy 확인 가능
- **AUC 제출 시**: `predict_proba(X_test)[:, 1]` 사용

## 주의사항

- city, gender, enrolled_university 등 범주형 많음 → 인코딩 필수
- 결측치: fillna(0) 또는 dropna() (데이터 특성에 따라)
