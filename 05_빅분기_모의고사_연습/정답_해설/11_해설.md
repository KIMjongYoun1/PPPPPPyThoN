# 11. Titanic 타이타닉 — 해설

## 풀이 흐름

1. train만 로드 (test 없음)
2. X/y 분리, 범주형(Sex, Embarked) 인코딩
3. train_test_split → 검증
4. 모델 학습 → 예측

## 핵심 포인트

- **stratify=y**: 분할 시 Survived 비율 유지
- test 없음: train_test_split으로 검증용 분할
- 실제 제출: DataManim에 test 제공되면 동일 방식으로 예측

## 주의사항

- PassengerId: train에 있음, test에 있으면 제출 시 사용
- Age 결측: fillna(0) 또는 median
