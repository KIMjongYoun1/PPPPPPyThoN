## 2. 1유형 — 집계·전처리

### 요구사항 → 사용 함수 → 답 (매핑표)

| 요구사항 | 사용할 것 | 결과 |
|----------|-----------|------|
| **A별 B 합계** (예: 나라별 골득점) | `df.groupby("A")["B"].sum()` | Series |
| **A별 B 평균** | `df.groupby("A")["B"].mean()` | Series |
| **A별 행 개수** | `df.groupby("A").size()` | Series |
| **상위 N개** (예: 상위 10개 채널) | `value_counts().head(10)` 또는 `sum().nlargest(10)` | Series |
| **가장 많은 값** (최빈값) | `value_counts().index[0]` | 값 1개 |
| **가장 큰/작은 행** (예: ratio 최대인 영상) | `sort_values("컬럼", ascending=False).iloc[0]` | 행 1개 |
| **조건 만족하는 개수** | `len(df[조건])` | 숫자 |
| **이름에 'X' 포함** (대소문자 무시) | `df["컬럼"].str.lower().str.contains("x")` | boolean |
| **요일별 개수** | `df["날짜"].dt.day_name()` + `groupby` | DataFrame |
| **A별 B별 개수 → 표** | `groupby(["A","B"]).size()` + `pivot` | DataFrame |
| **비율/ratio 구해서 정렬** | `df["ratio"] = A/B` → `sort_values("ratio")` | DataFrame |
| **채널명 바꾼 케이스** | `drop_duplicates(["A","B"]).B.value_counts()` → `>1` 개수 | 숫자 |
| **Years split 후 4자리 체크** | `str.split("-")` → `apply(체크함수)` | boolean |
| **N회 이상인 개수** | `(df[["A","B"]].value_counts() >= N).sum()` | 숫자 |
| **view_count=0 제외** | `df[df["view_count"] != 0]` (ratio 구할 때) | DataFrame |

## 3. 2유형 — 분류/회귀 파이프라인

### 요구사항 → 사용 함수 → 답 (매핑표)

| 요구사항 | 사용할 것 | 결과 |
|----------|-----------|------|
| **train/test에서 X, y 분리** | `drop(columns=["y","ID"])` | X_train, y_train |
| **X_train/X_test 이미 분리** | ID만 분리, `drop(columns=["ID"])` | X_train, X_test |
| **범주형 → 숫자** | `LabelEncoder().fit_transform` | 인코딩된 X |
| **결측 채우기** | `fillna(0)` | 전처리된 X |
| **분류 예측** | `RandomForestClassifier().fit().predict()` | 0/1 |
| **회귀 예측** | `RandomForestRegressor().fit().predict()` | 연속값 |
| **확률/AUC 제출** | `predict_proba(X)[:, 1]` | 0~1 |
| **제출 파일** | `pd.DataFrame({"ID": ids, "y": pred}).to_csv(..., index=False)` | submission.csv |
| **다중분류** | y도 LabelEncoder → `inverse_transform` | DrugY 등 |

---

### X, y 분리

```python
# train/test 있는 경우
y_train = train["타겟컬럼"]
X_train = train.drop(columns=["타겟컬럼", "ID"])
X_test = test.drop(columns=["ID"])
test_ids = test["ID"]

# X_train, X_test 이미 있는 경우 (HRdata, diabetes 등)
test_ids = X_test["ID"]  # 또는 X_test["enrollee_id"] 등
X_train = X_train.drop(columns=["ID"], errors="ignore")
X_test = X_test.drop(columns=["ID"])
y_train = y_train["target"]  # 컬럼명 확인
```

### 범주형 자동 탐지

```python
cat_cols = X_train.select_dtypes(include=["object"]).columns.tolist()
```

### LabelEncoder (범주형)

```python
from sklearn.preprocessing import LabelEncoder

for col in ["Geography", "Gender"]:   # 범주형 컬럼
    le = LabelEncoder()
    X_train[col] = le.fit_transform(X_train[col].astype(str))
    X_test[col] = le.transform(X_test[col].astype(str))
```

### 결측치

```python
X_train = X_train.fillna(0)
X_test = X_test.fillna(0)
```

### 분류 모델

```python
from sklearn.ensemble import RandomForestClassifier

model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(X_train, y_train)
pred = model.predict(X_test)
```

### 회귀 모델

```python
from sklearn.ensemble import RandomForestRegressor

model = RandomForestRegressor(n_estimators=100, random_state=42)
model.fit(X_train, y_train)
pred = model.predict(X_test)
```

### 제출

```python
submission = pd.DataFrame({"ID": test_ids, "타겟컬럼": pred})
submission.to_csv("submission.csv", index=False)
```

### 다중분류 (y도 인코딩)

```python
le_y = LabelEncoder()
y_enc = le_y.fit_transform(y_train)
model.fit(X_train, y_enc)
pred_enc = model.predict(X_test)
pred = le_y.inverse_transform(pred_enc)
```

## 4. 3유형 — 통계

### 요구사항 → 사용 함수 → 답 (매핑표)

| 요구사항 | 사용할 것 | 결과 |
|----------|-----------|------|
| **F검정 통계량** (분산 비) | `np.var(..., ddof=1)` → 큰분산/작은분산 | round(값, 3) |
| **합동 분산** | `((n1-1)*v1+(n2-1)*v2)/(n1+n2-2)` | round(값, 3) |
| **t검정 p-value** | `stats.ttest_ind(g1, g2)` | round(p_value, 3) |
| **오즈비** | `logit().fit()` → `np.exp(params["x"])` | round(값, 3) |
| **유의한 변수 회귀계수** | `pvalues < 0.05` → `params[sig.index]` | round(값, 3) |
| **R² (결정계수)** | `ols().fit()` → `model.rsquared` | round(값, 3) |
| **유의한 변수 회귀계수 합** | `params[pvalues<0.05].sum()` (Intercept 제외) | round(합, 3) |
| **로그 변환** (23번) | `np.log(값)` — F/t검정 전 | 변환된 시리즈 |

---

### F검정, t검정 (scipy)

```python
import numpy as np
from scipy import stats

# 1) 두 집단 분리, 로그 변환 (23번)
cls_1 = df[df["Classification"] == 1]["Resistin"]
cls_2 = df[df["Classification"] == 2]["Resistin"]
g1 = np.log(cls_1)
g2 = np.log(cls_2)

# 2) 분산 (ddof=1: 표본)
var1 = np.var(g1, ddof=1)
var2 = np.var(g2, ddof=1)
f_stat = max(var1, var2) / min(var1, var2)  # 큰/작은

# 3) 합동 분산
n1, n2 = len(g1), len(g2)
var_pooled = ((n1-1)*var1 + (n2-1)*var2) / (n1+n2-2)

# 4) t검정
t_stat, p_value = stats.ttest_ind(g1, g2)
```

### 로지스틱 회귀 (statsmodels)

```python
from statsmodels.formula.api import logit

# 전제: df에 target, x1, x2 있고 결측 제거 (logit은 NaN 불가)
df = df.dropna()
model = logit("target ~ x1 + x2", data=df).fit()
odds_ratio = np.exp(model.params["x1"])
sig = model.pvalues[model.pvalues < 0.05].drop("Intercept", errors="ignore")
```

### 다중선형회귀 (statsmodels)

```python
from statsmodels.formula.api import ols

# 전제: df에 y, x1, x2, x3 있고 결측 제거. 컬럼명에 공백 있으면 rename
model = ols("y ~ x1 + x2 + x3", data=df).fit()
r2 = model.rsquared
sig_coefs = model.params[model.pvalues < 0.05].drop("Intercept", errors="ignore")
coef_sum = sig_coefs.sum()
```

## 6. 자주 틀리는 것

| 실수 | 올바름 |
|------|--------|
| `fit`만 train에 | test는 `transform`만 (LabelEncoder) |
| 컬럼명 대소문자 | CustomerId vs customerId, ID vs id 확인 |
| submission index | `index=False` |
| 3유형 sklearn | statsmodels (logit, ols) |
| round(값, 3) | 소수 셋째자리 |
| ratio 구할 때 | view_count=0, like=0 제외 (division by zero) |
| logit/ols | 결측 제거 필수 (`df.dropna()`) |
| pivot columns | 실제 컬럼명 사용 (trending_date2 등) |
