## 🚀 빠른 시작

### 1. 가상환경 활성화
```bash
cd /Users/ryankim/Python
source my_python_env/bin/activate   # macOS/Linux
# Windows: my_python_env\Scripts\activate
```

### 2. 처음이라면 (Python·가상환경 없음)
```bash
# 0단계: Python 설치 → 01_가이드문서/01_python_setup.md
# 0단계: 가상환경 생성
cd /Users/ryankim/Python
python3 -m venv my_python_env
source my_python_env/bin/activate
```

### 3. 빅분기 목표라면 (권장 순서)
```bash
# Step 1: 03_기초부터_심화 01번부터 순서대로
python 03_기초부터_심화/01_변수와출력.py
python 03_기초부터_심화/02_리스트와반복.py
# ... 13번까지

# Step 2: 06_빅분기_필기_정리 읽기 (필기 대비)

# Step 3: 05_빅분기_모의고사_연습 풀기
python 05_빅분기_모의고사_연습/01_데이터_로드_연습.py
# ...
```

### 4. Python 문법만 먼저 하고 싶다면
```bash
# 01_가이드문서 + 02_실습코드 + 03_정답지
cat 01_가이드문서/02_python_basics.md
python 02_실습코드/02_practice.py
python 03_정답지/02_answer.py
```

## ⚙️ 자동완성 비활성화 (실기 대비)

`.vscode/settings.json`에서 **자동완성을 비활성화**해 두었습니다.

| 항목 | 내용 |
|------|------|
| **이유** | 빅분기 실기는 구름(goorm) IDE에서 진행되며, 자동완성이 거의 없음 (쌩코딩에 가까움) |
| **목적** | 시험장과 동일한 조건에서 연습 → 함수·문법을 직접 기억하고 타이핑하는 습관 |
| **설정** | `editor.quickSuggestions`, `editor.suggestOnTriggerCharacters` 등 false |
| **복원** | `.vscode/settings.json`에서 해당 설정 삭제 또는 `true`로 변경 |
