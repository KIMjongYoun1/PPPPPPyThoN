# 기본 개념 실습 - 사용 방법

## 학습 로드맵: 기초 → 어려운 난이도 (추천 순서)

정확히 이해가 안 되어도, **아래 순서대로** 한 단계씩 해 보면 기초부터 심화까지 쌓을 수 있음.

| 단계 | 난이도 | 할 일 | 쓰는 파일 |
|------|--------|--------|-----------|
| **1** | ★☆☆ 기초 | 따라만 치고 실행해 보기 | `practice_follow_along.py` (A~D편) |
| **2** | ★☆☆ 기초 | 개념 읽고 따라 치기 | `learn_concepts_by_typing.py` **1~8단계** (Part 1만) |
| **3** | ★★☆ 중급 | 빈칸 채우기 (할 수 있는 것만) | `practice_basic_concepts.py` 실습 1, 2, 6 → 3 |
| **4** | ★★☆ 중급 | 파일 읽기·열 추출·null→평균 채우기 흐름 익히기 | `learn_concepts_by_typing.py` **6단계** (6-1~6-6) |
| **5** | ★★★ 심화 | pandas 집계·구간화·가설검정 | `learn_concepts_by_typing.py` **9~16단계** |
| **6** | ★★★ 심화 | 가설검정 원리(말로) + t/카이제곱 코드 패턴 | `hypothesis_test_visual.py` (원리 설명 + Part 1, 3만 읽고 실행) |

**팁:** 1→2→3까지 완료한 뒤, 4에서 "파일·열·결측" 흐름을 한 번에 이해하려고 읽어 보기. 5~6은 시험 직전이나 강의 들은 뒤에 다시 보면 더 잘 들어옴.

## 0. 직접 치면서 개념·메서드·문제 풀이 학습

- **`learn_concepts_by_typing.py`** 를 연다.
- **1단계 ~ 8단계**까지 순서대로:
  - **[개념]** / **[메서드]** 설명을 읽고,
  - **따라 칠 코드** 블록을 직접 타이핑한 뒤 실행해 본다.
- 변수, f-string, 리스트, 슬라이스, for, append, 리스트 컴프리헌션, strip/split, 파일 읽기, def, pandas 등 **문제 풀 때 필요한 내용**을 한 번에 익힌다.

## 0-2. 파이썬 처음이면: 따라치기

- **`practice_follow_along.py`** 를 연다.
- **A편 ~ D편** 코드를 한 줄 한 줄 그대로 따라 친다 (복붙 말고 직접 타이핑).
- 저장 후 실행: `python practice_follow_along.py`
- 출력이 나오면 성공. 그다음 "빈칸 채우기"로 넘어가면 됨.

## 1. 할 일 (빈칸 채우기)

- `practice_basic_concepts.py` 파일을 연다.
- 각 **실습**에서 `None` 이나 "여기에 작성" 부분을 **스스로 코드로 채운다**.
- 아직 안 배운 부분은 비워 두어도 된다. 할 수 있는 것만 채워도 됨.

## 2. 실행

터미널에서:

```bash
cd 02_실습코드
python practice_basic_concepts.py
```

## 3. 검사 받기

- 실행하면 맨 아래에 **=== 검사용 출력 ===** 이 나온다.
- 그 **출력 전체**를 복사해서 채팅에 붙여넣고, **"검사해줘"** 라고 보내면 된다.
- 그러면 답이 맞는지, 어디를 고치면 좋을지 알려줄 수 있다.

| 실습 | 내용 | 난이도 |
|------|------|--------|
| 1 | 변수, f-string | ★☆☆ |
| 2 | 리스트, for | ★☆☆ |
| 3 | CSV 읽기 (open + split) | ★★☆ |
